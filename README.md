# AI PM Prompt Library

## Overview
This repository contains reusable AI prompt frameworks designed to support program management, incident response, migration planning, risk analysis, RCA generation, and stakeholder communication.

## Purpose
The goal of this library is to turn AI into a practical operational tool for real-world program and project management workflows.

## Prompt Categories
- Migration Planning
- Risk Management
- Incident Intake and Response
- RCA Generation
- Stakeholder Communication
- Workflow Improvement
- Task Breakdown
